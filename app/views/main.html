Grid
Lightweight, fast & LÖVE-ly.

Grid is a 2D game engine built by Planimeter.
It was designed for our fantasy MMORPG, Vertex Adventure.

Examples

Grid lets you jump straight into game logic.

Making a HUD element

class "health" ( gui.panel ) function health:draw() graphics.print( localplayer:getHealth() ) end

Making an entity

class "monster" ( npc ) function monster:monster() self:setSprite( "images/monster.png" ) end

Playing a sound from an entity

function monster:takeDamage() self:emitSound( "sounds/cry.wav" ) end
